# CryptX Dashboard

A cryptocurrency analytics dashboard built with React and Tailwind CSS.

## Features

- Sidebar with navigation menu
- Cryptocurrency price cards
- BTC price chart with SVG
- Live market data table
- Transactions history table
- Responsive design

## Tech Stack

- React
- Tailwind CSS
- SVG for charts
- JavaScript
